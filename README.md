# 🏦 Bank Loan Analysis Project (Python)

## 📌 Project Overview
This project focuses on **Bank Loan Analysis using Python** to evaluate loan applications, funded amounts, repayments, and loan performance trends.

The objective is to generate meaningful business insights from loan data and help financial institutions understand:
- Loan application trends
- Funded vs received amount
- Good and bad loan performance
- Interest rate and DTI analysis
- Regional and monthly trends

This project is ideal for showcasing **Python data analysis, reporting, and financial analytics skills** on GitHub.

---

## 🎯 Business Problem
Banks receive thousands of loan applications and need to monitor:
- Total number of applications
- Total amount funded
- Total amount received
- Loan default trends
- Regional performance
- Month-on-month growth

This analysis helps support **better lending decisions and risk management**.

## 📂 Dataset Information
The dataset contains loan-related information such as:
- `id` → Loan application ID
- `issue_date` → Loan issue date
- `loan_amount` → Amount funded
- `total_payment` → Amount received
- `int_rate` → Interest rate
- `dti` → Debt-to-income ratio
- `loan_status` → Good / Bad loan classification
- `address_state` → State-wise loan data
